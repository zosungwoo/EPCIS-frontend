Instruction for the Capture Client

Method Name Description

 The services usable in this page allow providing query results.
  • Poll: Return events/vocabularies stored in the EPCIS repository in interest immediately (For various query options, see EPCIS Section 8.2.7.1 and 8.2.7.2)
  • GetStandardVersion: Return the standard version of EPCIS
  • GetVendorVersion: Return the vendor specific version of EPCIS
  • GetQueryNames: Return a list of all query names available
Note: Subscribe/Unsubscribe/GetSubscriptionIDs are supported in the other module.

Method: POST
Contents: application/xml, complying with EPCIS WSDL

How to use:
  • Load an example in a dropdown menu or Copy/paste/text your own EPCIS query into a textbox
  • Click 'Query' and See the results
